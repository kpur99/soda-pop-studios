New Building. 100-Year-Old Interior.

Built in the 1920s as a Coca-Cola bottling plant, the building has been completely gutted — new electrical, new plumbing, new HVAC, new everything. Only the original brick remains, which is exactly the point.

Corner lot. Elevator access. Second-floor event space with outside deck. ADA compliant. Sprinklers throughout.
